# ci-blog

#### Installation:    
$ git clone --depth=1 https://github.com/sudogem/ci-blog.git    
$ cd ci-blog/    
$ npm install   
$ npm install gulp   
$ npm install semantic-ui    
$ cd semantic/    
$ gulp build    


#### Technology stacks:
* Node.js v 8.1.0
* Codeigniter 3.1.3    
* Codeigniter-HMVC (https://bitbucket.org/wiredesignz/codeigniter-modular-extensions-hmvc)
